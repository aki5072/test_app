# あなたの「タグ」は増えていますか？

![eyecatch](https://assets.st-note.com/production/uploads/images/166105268/rectangle_large_type_2_6c0ea23a2f1f66273b28d68e32ddb161.png?fit=bounds&quality=85&width=1280)

---

毎年この季節になると**「今年は去年よりも成長できただろうか？」** と考える機会が増えます。

ちなみにはやしの場合は、今年よりも昨年の方が**「成長できた！」** という実感がありましたね。転職、東京→福岡への移住、同棲、副業でマーケ支援を行う、月収100万円達成など…。

ライフイベントの変化があれば、その分**「成長した！」** という実感がありますが、今年は昨年の延長線上で生きている感覚があり、正直大きく成長した、という実感は薄いです（これは今年の反省ですね）。

今回伝えたい事は**、「あなたの"タグ"は毎年増えていますか？」** ということです。

ちなみにここで言う**「タグ」** とは、

> **・スキル  
>  ・数字として表せられるような成果**

こういったものが挙げられるかな、と。

要は理想の人生やキャリアを手に入れたいなら、それ相応のタグを保有しておかないと叶えられないよ、ってことです。

例えばはやしの場合は、これまで持っていたタグとして

> SNSマーケティング（10,000フォロワー超えのアカウント複数運用） / X運用 / インスタ運用 / YouTube運用 / Webマーケティングの戦略立案 / Meta広告運用 / リファラルマーケティングの戦略立案〜運用 / 10人規模のチームマネジメント

みたいなものが挙げられますが、昨年秋〜本業では事業部のマネージャーとして事業開発を担う立場になったので、**「事業開発の基礎基本」** みたいなものが、新たなタグとして増えた感じでしょうか。

ちょうど直近で事業開発の職種の求人をよく見ていたのですが、そこで求められるスキルとしては、

> ・マーケティングの戦略立案〜実行経験  
> ・P / L（損益計算書）の策定〜運用の経験  
> ・チームマネジメントの経験  
> ・BIツールの運用、SQL等の使用したデータ分析の経験

このあたりが事業開発の職種に応募する際に必須〜歓迎スキルとして記載があったので、仮にはやしがマーケティング職→事業開発職に行くなら、今はタグとして保有していない**「BIツールの運用」** などが、新たに学ばなければいけない項目として導き出されます。

ちょっと話は逸れますが、例えば転職する際に**「こんな職種に就きたい」** とう理想があれば、その職種で求められる**必須条件や歓迎条件をチェック** して、**「今自分に足りないスキル」** を今の会社で磨くなりすると、キャリアアップのハードルは下がるかな、と。

はやしも来年で30歳になりますし、会社員としてのキャリアを広げるためにも、まずは今の会社で**「事業開発」** という分野をより磨いていこうかな、と思っています。

みなさんも**「理想とする人生、キャリア」** があれば、ぜひ**「タグを増やす」** と考えを持っていただくと、より理想に近づくので年内に**「今自分が持っているタグは何か？」** を振り返ってみて、来年の目標を立ててみてください。
