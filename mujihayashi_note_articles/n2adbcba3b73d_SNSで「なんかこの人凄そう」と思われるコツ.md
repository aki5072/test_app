# SNSで「なんかこの人凄そう」と思われるコツ

![eyecatch](https://assets.st-note.com/production/uploads/images/201508087/rectangle_large_type_2_c1b43710d750ae8625db2d9fc577d29a.png?fit=bounds&quality=85&width=1280)

日頃からXやnoteを眺めていると、

**「なんかこの人凄そうだな」**

と思ったことはないでしょうか。その人が本当に凄いのか、それとも弱小なのか、は実際に会わないとわからないですが、SNSでは

**「リアルでは全然凄くないのに、SNS上では凄そう」**

と思われている人が一定数います。

ってことで、今回はこの**「なんかこの人凄そう」** と思ってしまう理由を解説しますが、今回お伝えすることを応用すれば、もちろんあなたの発信にも役立つ内容になります。

実際、SNSは**「なんかこの人凄そう」** と思われた者勝ちな節はありますからね。何かしら実績があるなら尚更です。

**「なんかこの人凄そう」** と思ってしまう理由。それは、

**その発信者が「錯覚資産」を上手に活用しているから、です。**

ここで新たなキーワードが出てきましたね。**錯覚資産とは、ある一定の分野で成果を出している人を、他分野でも「凄そう」と錯覚してしまう現象** のことを指します。

詳しくは以下の書籍をぜひ読んでいただきたいのですが、

[ **人生は、運よりも実力よりも「勘違いさせる力」で決まっている** __ _amzn.to_ _1,640円_ (2025年07月11日 09:23時点 詳しくはこちら) Amazon.co.jpで購入する ](https://amzn.to/4697x8B) [ ](https://amzn.to/4697x8B)

私も7年前、新卒4ヶ月で会社を辞める→半年でXのフォロワーを1万人まで伸ばした時には、この書籍に書かれている**「錯覚資産」** を上手に活用した過去があります。

先ほどの説明だとわかりづらいと思うので、具体的な錯覚資産を例に出すと**「月収7桁です」とか「早稲田卒です」とか「上場企業の部長です」とか「社長です」みたいな、見るからに「なんかこの人凄そう」という経歴** は最たる例でしょう。

よくXではプロフィールに**「月収7桁です」** と書いている人がいますが、あれってただ自慢したいわけではないんですよね。

**月収7桁です、という事実をプロフィールに書く→なんかこの人凄そう、という錯覚資産になる**

というカラクリを使っているだけです。

また、こういうのもありますよね。アカウント名が**「〇〇社長」** みたいな、社長と自分から名乗っているパターン。

これも錯覚資産の一例です。その人の会社が赤字経営でも、年間の売上が100万円だったとしても、「社長」と付くだけで**「なんかこの人凄そう」** って人は錯覚してしまうんです。だからフォロワーが集まる、というカラクリ。

「なんでこんな匿名アカウントで、どんなビジネスをしているかわからない人がフォロワー数万人いるんだろう」

と疑問を抱いた方も、今回の錯覚資産さえ理解できれば、人間はそんな単純な実績を書かれただけで「凄そう」と錯覚→フォローしてしまう、というカラクリをご理解いただけたかと思います。

ちなみに私の発信でも、この錯覚資産を上手に活用しています。「noteで飯を食っていく」のプロフィールを見てみると、

![](https://assets.st-note.com/img/1752193802-LDnPMVe5RQ0Xihlvdr7FmJzN.jpg)noteで飯を食っていく、のXプロフィール

ここまでお読みいただいたあなたなら、どこに錯覚資産が仕掛けられているかがわかるはずです。

**「X1.5万人」「インスタ2万人」「YouTube5.5万人」「マーケコンサル企業のマネージャー」「メンバーシップは2週間で50人増えた」**

これ全て事実なんですけど、noteで飯を食っていく、という発信のコツをテーマにしたアカウントで、この実績を見てしまうと「なんか凄そう」となるはずです。

実際、このアカウントは1ヶ月ちょっとでバズなしでフォロワー600人まで増やしていますからね。論より証拠です。

とはいえ、ただプロフィールに実績を書くだけでは「虚像では？」と思われちゃうでしょう。そこで日頃のポストでは

**「あ、この人ホンモノだ」**

と思わせるようなあなたの知見を出していくんです。私だったら、

> noteで飯が食える家、というのはこのレベルまで持っていくことです。白床の開放的な空間に、無印良品で揃えた家具家電。家中の持ち物全部「無印にしました！」くらいの狂人さがないと、暮らしの発信で飯は食えないでしょう。 [pic.twitter.com/GDXipADcPe](https://t.co/GDXipADcPe)
> 
> — noteで飯を食っていく (@note_hayashi) [June 27, 2025](https://twitter.com/note_hayashi/status/1938720417272996354?ref_src=twsrc%5Etfw)

上記ポストでは**「この人は本当に暮らしの発信で飯食ってるんだ」** という権威生につながりますし、

> 20万円の副業収入を1つで稼ぐのは大変ですが、2万円の仕事を10個やれば20万円稼げます。みんな「1つで20万円」目指そうとするんですけど、最初からそれ目指すの無理ゲーです。私なら2万円の仕事を10個作って、その過程をnoteに書いて有料で出して、また稼ぎを増やします。令和の百姓を目指しましょう。
> 
> — noteで飯を食っていく (@note_hayashi) [July 9, 2025](https://twitter.com/note_hayashi/status/1942925436004028440?ref_src=twsrc%5Etfw)

直近伸びたこのポストは、私が実際に複数の副業をこなしているからこそ、伸びている、とも言えるでしょう。
