# mixi2でフォロワーを増やす方法

![eyecatch](https://assets.st-note.com/production/uploads/images/166230624/rectangle_large_type_2_9eeb5adc42606052b95287bf339be342.png?fit=bounds&quality=85&width=1280)

一昨日くらいですかね、15〜20年ほど前に一世を風靡したSNS、「mixi」が新たなSNSをリリースしました。その名も**「mixi2」** 。

あ、まだmixi2入れてない方は招待制なので、ぜひ以下から登録してみてください。

[ **[mixi2への招待] シンプリストはやしさんとはじめよう** _「無印良品で叶える"ちょっといい暮らし"」をテーマに、YouTubeで無印料理レビューやシンプルな暮らしを発信しています！_ _mixi.social_ ](https://mixi.social/invitations/@muji_hayashi/S6wkS4RB3Te9bSug1oQ9Tx)[](https://mixi.social/invitations/@muji_hayashi/S6wkS4RB3Te9bSug1oQ9Tx)

残念ながらはやしはmixi世代ではなく、その後の**「モバゲー」「GREE」世代** になります。

当時を思い返すと、GREEのプロフィールに「since〜」みたいに書いて付き合っている事を匂わせたり、当時どハマりしていたHKT48のファンコミュニティに入って、コミュニティに入っている方と一緒に劇場公演とか見に行ったのが懐かしいです（もはや黒歴史に近いが）。

あれから10数年が経ちましたが、ここに来て**「コミュニティ」** を重視する国産SNSが登場するのは非常に面白い流れだな、と思っていて。

XやInstagram、TikTokといった外国産のSNSは、自分がフォローしているユーザーの投稿よりも**「おすすめ」** でレコメンドされるコンテンツで溢れていて、ツールベンダーも**「いかにSNS上での滞在時間を長くさせて、広告をクリックさせるか？」** しか正直考えていないんですよね。

その影響もあり、**自分がフォローしているユーザーの投稿＜レコメンドされた投稿の方がよく流れる** 、というロジックです。

しかし、mixi2にレコメンド機能はなく、むしろ**「自分がフォローしたユーザーの投稿」「自分が所属しているコミュニティの投稿」** しか表示されないので、ある意味**「自分が欲しい情報だけ、厳選して見ることができる」** というメリットがあります。

mixi2ブームがいつまで続くかは不明瞭ですし、はやしも別に運用する気はないのですが…マーケターとして、**「mixi2内でフォロワーを伸ばす方法」** がなんとなく分かったので、ここに残しておこうかな、と。

結論から言うと、mixi2でフォロワーを伸ばす方法は、

**自分が興味・関心のあるコミュニティに所属し、コミュニティ内で活発に動く事で「その界隈の盟主」になることです。**

おそらく9割の方がmixi2に懐かしさを感じているのは**「コミュニティ機能」** だと思っていて。

自分の趣味とか、仕事のつながりとか、興味関心のあるコミュニティに所属して、情報を発信したり、**「見る専」** として情報収集を行うユーザーが大半です。

例えば、はやしの場合は**「無印良品」** のコミュニティに所属して1つ投稿してみましたが、一定数反応が得られましたし、今後無印良品のコミュニティで有益な情報（おすすめアイテムや、意外な使い方とか）を発信していけば、自然と**「お、この人ってめちゃ有益な情報発信してる…フォローしておこう」** とユーザー心理としてはなるはずです。

  


![](https://assets.st-note.com/img/1734510118-pFo5KrM19xz2d6QhXSjECecV.jpg)はやしがmixi2内の無印良品のコミュニティに投稿した内容

他にもサウナのコミュニティにも入って見ましたが、例えば都内のサウナを1日1回、そのコミュニティに発信してリアルな感想を投稿すれば、それだけでも有益な情報になりますし、フォローされる確率は上がります。

要はmixi2というSNS全体でインフルエンサーになるのではなく、**自分の興味・関心のあるコミュニティで「盟主」的な存在になることが、mixi2でフォロワーを増やす効果的な方法な** のかな、と。

また、フォローされる上で大事なのは**「プロフィールの設計」** です。例えば無印良品のコミュニティに所属しているユーザーからフォローをされたいなら、**「無印良品を紹介してます！」** と、コミュニティのユーザー属性と情報を揃える必要があります。

![](https://assets.st-note.com/img/1734509778-zaqWiuMStPLBfw96sFxJeRA4.jpg)はやしのmixi2のプロフィール画面

Web広告で言うと、**クリエイティブとその遷移先のLPの訴求を揃える、と本質は同じ** ですね。要はカスタマージャーニーをしっかり作る（無印良品なら、無印良品と一貫性を持たせる）ことが大事です。

まあ、はやしはXとnote、YouTubeに専念すると決めたので、mixi2の熱が冷めないうちはコミュニティに投稿しようと思いますが…まあブームが過ぎ去れば、自然と見る専になるでしょう。

もしmixi2でフォロワー伸ばしたい、とか興味があればぜひやってみてください。
