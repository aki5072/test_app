# 自分の「強み」を見つける超簡単な方法

シンプリストはやしです。

楽しかった年末年始の9連休が終わり、仕事始めの方は多いのではないでしょうか。ちなみに昨日はこんなニュースが話題になりましたよね。

巷で話題の**「退職代行サービス」** ですが、やっぱり年末年始に仕事のことを考えて、**「私の居場所はこの会社ではない！」** と決心した方が、1/6に退職代行サービスを活用しまくったみたいですね。

今日の本題にも繋がりますが、**「会社を辞めたい」** と思う理由として、人間関係が筆頭に上がりますが、次に多いのが**「今やっている仕事が自分に向いていない」** という葛藤です。

そこでよく言われるのが**、「自分の強みを活かした仕事をやろう！」** ですが、**「いやいや、そう簡単に自分の強みなんて見つからないよ…」** と悩む方が大多数だと思っています。

**しかし、「自分の強みを見つける方法」は意外と簡単なんです。**

先ほど、私がキャリアアドバイザーとして支援している求職者の後輩とやりとりをしていたのですが、

**自分にとっては「当たり前」だと思って取り組んでいることや、その行為に対して苦痛と感じていないことが、あなたの「強み」になります。**

先ほどお話しした後輩に関しては、現職では法人営業をやっているのですが、商談の議事録をコピーして、**お客様目線で「もっと良い提案はできないだろうか？」と考えることが得意** だったんですよね。

正直、LINEが送られた時は衝撃でした。

さすがにその資料をお見せすることは難しいのですが…議事録を紙にコピーして、お客様が会話した内容に対して**「こんな提案ができそうだ」** とか、**「次の商談ではこんなコンテンツを用意して持っていこう」** など、とにかくお客様に対してのフォローをすんなり考え、メモを取りまくっていました。

**本人には「これが君の強みだよ！」と伝えると、「え、こんなこと誰でもやってるんじゃないですか？」と返答が。**

いやいや…営業もピンキリですからね。お客様のことなんて何も考えずに、ただ商材を売りつけるような方もいる中、**「お客様目線」** で、**「お客様がより成果を出せる提案」** ができるのは、まさに営業の見本だな、と感心しました。

こんな感じで、**自分にとっては「当たり前」にできていることでも、他人は「できない」こと…それがあなたの強み** になります。

後輩の場合は、今はゴリゴリの営業会社に勤めているので、**「お客様目線で提案する」** という強みが活かせていないんですよね…転職活動では、もし営業を続けるなら提案型の営業ができるような企業に推薦を上げています。

ちなみにはやしの場合も、毎日1,000〜2,000文字のnoteを20分くらいで執筆することは苦ではないのですが、先日YouTube仲間と飲みに行った際に「それがはやしさんの強みですよ」と言われたんですよね。

こんな感じで、**自分では「努力」だと思っていない行為でも、他人は「努力しないとできない」行為も強み** になります。

今の職場でくすぶっていたり、転職を考えている方は、仲の良い友達や、相談できる先輩、メンターに**「私の強みってなんだと思いますか？」** と聞いてみると、案外すぐに見つかるでしょう。
