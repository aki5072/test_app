# SNSでオワコンにならない方法

![eyecatch](https://assets.st-note.com/production/uploads/images/201929247/rectangle_large_type_2_01aa61358ec5ed2f053738c729bb2a56.png?fit=bounds&quality=85&width=1280)

**「そういえばあの人、最近Xで見かけなくなったよね」「あのアカウントはもう"オワコンだね」**

普段からXを見ていると、ふと気づいた時に**「そういえばあのアカウント、最近見なくなったなぁ」** ってことはないでしょうか。

はやしもあります。私は2018年〜SNSで情報発信をやっているので、これまでも数多くの**「急激にバズってフォロワーを増やしたものの、気づいたら見なくなっていた」** アカウントは思い返すとキリがないです。

もしあなたが「SNSを副業にしたい」「Xとnoteを掛け合わせて収入の柱としたい」と考えているのであれば、

**「SNSでオワコンにならない方法」**

というのは一定の需要があると思っていて。実際にSNSでの情報発信歴が８年目に突入＝**オワコンになっていない** 、私の知見をお伝えしようかな、と。

結論、SNSでオワコンにならない方法はこれだけです。その方法とは、

**1つのSNSで成果を出す（目安はフォロワー1万人）→他のSNSを伸ばしていく→他のSNSを伸ばした過程を、また別アカウントを開設して発信する**

私の場合、2018年の8月に、新卒4ヶ月で会社を辞める→その後すぐに、フリーランスのブロガーとして独立しました。

その際に、ブログに集客することを目的にXを開設→2019年の5月にフォロワー1万人を達成したのですが、もしこのまま「ブログとXだけ」を主軸にマネタイズをしていたら、情報発信からは消えていたと思っています。

これは運要素もありますが、私がXのフォロワー1万人を達成した直後に取り組んだことは、YouTubeでの発信です。

[ **シンプリストはやし** _「無印良品で叶える"ちょっといい暮らし"」をテーマに、無印良品のレビューをはじめシンプルなモノ・ガジェット・暮らしのアイテ_ _www.youtube.com_ ](https://www.youtube.com/channel/UCYoG_ZlQMDyMaeg7pLWsN2Q)[](https://www.youtube.com/channel/UCYoG_ZlQMDyMaeg7pLWsN2Q)

現在は登録者5.5万人いるチャンネルになっていますが、YouTubeではフリーランスブロガーとしての切り口ではなく、全く異なる切り口で、しかも「暮らし系」の発信を始めました。

これが私が「無印良品」のレビュー系YouTuberになったきっかけです。ありがたいことに、2019年の11月にYouTubeも登録者1万人を達成し、その頃にはXがフォロワー1万人、YouTubeも登録者1万人と、複数の発信媒体でフォロワー1万人を達成させています。

しかし、ここでもYouTubeだけに固執していたら、またまた消える危機に直面していたと思っていて。2020年ごろから、今度はInstagramにおいて「無印良品系の発信者」が増えてきたんですよね。

実際にフォロワー10万人を超えるインフルエンサーがひしめく中、私も重い腰を上げて2021年〜はInstagramに注力しました。

現在はXとnote、YouTubeに注力する、と決めているのでInstagramの発信はやめていますが、2年ほどでフォロワー2万人まで伸ばすことができ、リール（ショートムービー）でも100万回再生を超えるコンテンツを作るなど、Instagramでも「シンプリストはやし」の名前を一定広めることができています。

ちなみに、**「noteで飯を食っていく」** のアカウントも、私が今度は**「Xとnoteで飯を食っている人、というポジションを確立したい」** というゴールのもと、取り組んでいます。

このように、1つのSNSで成果を出す→そこで学んだこと、SNS運用の本質を活用して、他のSNSを伸ばす→再度Xに戻り、別アカウントを開設して「SNSの伸ばし方」でもポジションを取っていく

これが直近の私の戦略です。正直**「もっと早めにやればよかった」** と後悔しているレベルですが、おかげさまでnoteで飯を食っていく、のアカウントも1ヶ月ちょっとで650フォロワー、このメンバーシップも約80名の方にご登録いただいていますので、初動は成功しています。

1つのSNSを極めることができれば、そこで学んだ知見を他SNSでも活用できます。Xを一定伸ばすことができた人は、今度はThreadsやInstagramなど、ぜひ裾野を広げてみてください。
