# 理想の人生を送るために、「余白」を意識する

![eyecatch](https://assets.st-note.com/production/uploads/images/169364705/rectangle_large_type_2_21211444237a7764935d3655b3218e3d.png?fit=bounds&quality=85&width=1280)

シンプリストはやしです。

2025年が始まって早1週間が経過しましたが、年始に立てた目標とか、**「2025年はこんな1年にしたいな〜」** など、理想に向けて動き出せていますでしょうか。

あるあるなのが、こういった目標って、1月は**「頑張ろう！」** って意識できてるんですよ。2月、3月…と、時が経つといつの間にか忘れてるんですよね。

そんな時に意識してほしいのが、今回紹介する**「余白」** です。余白と聞くと、目に見える形で**「真っ白」** を意識すると思いますが、内面的な「余白」も存在します。

> **目に見える余白：掃除 / 整理整頓 / 余白のある空間を作る、など  
>  目に見えない余白：精神的な余裕 / 時間的な余裕、など**

ざっくりこんな感じでしょうか。はやしは**「シンプリスト」** として発信しているので、シンプリストたる者、常に**「余白」** を意識した生活を送っています。

今日お伝えしたいのは、**「余白」を意識した生活を送ることで、理想の人生に一歩近づける** 、ということでしょうか。

その理由としては、

**理想とする人生を思い描く→注力するべきこと、しなくてもいいことを仕分ける→余白が生まれる→その余白の中で、新たなアイデア、取り組みが生まれる→理想とする人生が手に入る**

流れとしてはこんな感じですが、ちょっと難しいですよね。具体的なエピソードをご紹介します。

今回着目する**「余白」** は、目に見える余白ではなく**「目に見えない余白」** です。それは精神的な余裕であったり、時間的な余裕であったり、常に自分の中で、**「考え、アイデアがひらめきやすい余裕を作ること」** です。

私の場合は、1日の中で

> ・本業：事業開発、売上チェック、クライアント支援、マネジメント  
> ・副業：YouTube、note執筆、クライアントへのマーケ支援、人材支援

と、幅広い業務をこなしていますが、**単純作業はAIに任せたり、後輩指導のために仕事を振ることで、時間的余裕＝余白を生み出しています。**

全部自分でやってしまうと、さすがに1日24時間あっても足りません…そこをテクノロジーや人の手を借りることで、時間的余裕だけではなく、**「仕事でパンパンにならない」** という精神的余裕を手にしています。

これは先程お伝えした、**「やるべきこととやらなくてもいいことを決める」** に該当しますよね。仕分けることで、余白が生まれます。

実際に仕事を19時に切り上げることで、それ以外の時間はまさに**「余白から生まれた時間」** と言っても良いでしょう。ジムで運動したり、サウナに入ることで、理想とする人生を叶えるために取り組むべきことや、新たなアイデアがひらめくんです。

後は、そのひらめいた取り組むべきこと、アイデアを実行に移すだけです。時間的余裕を手にしているので、取り組むのも苦ではありません。

こんな感じで、日常の中で**「余白を作る」** を意識できれば、理想とする人生を叶えるためにやるべきこと、アイデアがひらめき、それを実践に移して、結果として理想とする人生が手に入る、というサイクルです。

もし1日が仕事でパツパツで、余白すら生み出せない環境であれば、AIなどテクノロジー活用や、人の手を借りるとか…それすら難しいのであれば、「転職」も視野に入れましょう。
