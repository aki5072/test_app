# 「何者かになりたい」という考えは捨てなさい

![eyecatch](https://assets.st-note.com/production/uploads/images/185500832/rectangle_large_type_2_41ee92e9ca7bf2aad2086e01c176c749.png?fit=bounds&quality=85&width=1280)

シンプリストはやしです。

私のメンバーシップを登録いただいている方の中には、**「SNSのフォロワーを増やしたい！」** と思っている方が一定数いらっしゃると思います。

では、ここで問いたいのですが、なぜSNSのフォロワー数を伸ばしたいのでしょうか。SNSで有名になりたいから？SNSを副業にして稼ぎたいから？

理由は10人いれば10通りの答えが帰ってくると思いますが、ここで大事な視点は、**「SNSのフォロワーが増える」＝「ネットで有名になる」という事実** です。

そう考えると、SNSを伸ばしていく、ということは、つまり**「何者かになる」** と同義になりますよね。例えば私の場合**「シンプリストはやし」** と名乗っているわけですから、シンプリスト界隈では一定知名度はある方だと思っています。

しかし、私はSNS歴7年目に突入しようとしていますが、そもそも「有名になりたい、何者かになりたい」と思ってSNSを始めたわけではありません。

今日のタイトルにもありますが、そもそも**SNSを伸ばしたいのであれば「何者かになりたい」という考えは早々に捨てるべき** 、と考えています。

その理由は

**「何者かになりたい！」という目標には、ゴールがないからです。**

そもそも「何者かになる」というのは、数値化するとどのレベルなのでしょうか。特定のジャンルで有名になること？それとも老若男女から支持されること？YouTubeならヒカキンレベルまでになること？

こう考えると、「何者かになりたい」という目標は定性的すぎて、数値目標にできない、という課題が生まれます。

もしあなたがSNSを伸ばしたい、と考えるのであれば、「何者かになりたい」みたいなふわっとした目標はゴールが見えず、モチベーションは低下する一方です。

一方で、**目標にするべき指標としてはやはり数値化できること、それも「売上」など、お金に変わる目標** を立てた方がうまくいきます。

例えば私の場合、元々SNSを始めたキッカケは「ブログで稼ぎたい」という欲求でした。

以下noteにも記載していますが、新卒4ヶ月で会社を辞めた後、ローリスク、ハイリターンで稼げるのがブログであり、そのブログの流入数を増やすために始めたのがSNS（Twitter）だった、というだけです。

もちろん、フォロワー数も目標にしていましたが、これも「何者かになりたい」と同義で、フォロワー数は青天井なので、仮にフォロワー1万人を目標にしても、2万、3万と上には上がいるので、キリがありません。

SNSを始めたい！という根本的なキッカケは「稼ぎたい」しかないと思うので、目標の置き方としては**「SNSでいくら稼ぐか？」** で良いです。

ブログと組み合わせるのであれば、アフィリエイトで月10万円稼ぐために必要なSNSの露出数から逆算してポストするとか、そういった具合で行動量が決まります。

有名になりたい、みたいなフワッとした目標の場合、「投稿をバズらせないと…」みたいな的外れの行動に走ってしまいますからね。大事なのは「SNS経由でいくら稼ぎたいか？」であり、バズることではありません。

売上から逆算した行動ができれば、必然的にSNSも伸びていくと思っています。当時の私がそうであったように。

結局有名になりたい！みたいなモチベーションではなく、「いくら稼ぎたいか？」をモチベにして、結果として**「あ、気づいたら界隈で有名になってたわ」** くらいを目指しましょう。その方が変に有名人を気取らなくても済むので。
