# 「言語化力」こそ、シンプルライフを叶える必須スキルな理由

シンプリストはやしです。

シンプルライフ、と一言で表しても、捉え方は人それぞれだと思っています。

ある人は、白を基調とした空間に、余白のある部屋でポツンと黒い家具があるような、ミニマリスト的なインテリアを想像するでしょう。

また、ある人は生き方そのものにフォーカスし、日常におけるあらゆる物事を効率化し、自分にとって本当に好きなことに注力できる時間を捻出する人を想像するでしょう。

ちなみにはやしが提唱するシンプルライフは後者になるのですが、同じ単語を見ても、人によって捉え方は千差万別なわけです。

つまり、「シンプルな暮らしがしたいな〜」という願望があるのであれば、そもそも自分にとっての「シンプルライフ」を定義しなければ、理想とする暮らしは手に入りません。

そこで必要なのが、**自分の思考を言葉で表現する力＝言語化力** です。

今回は日頃から言語化力を鍛える方法を紹介していきたいのですが、やるべきことはたった1つです。それは、

**「なぜそう思ったのか？」を思いつくだけ書き出しましょう。→これだけです。**

言語化力は、論理的思考能力があってこそ成り立つものであり、論理的思考能力は、具体と抽象を行き来することで鍛えられるものだと考えています。

ちょっと難しい言葉が重なってしまいましたが、要は、ふと思った自分の考えに対して、**「なぜそう思ったんだろう？」** を思いつくだけ書き出してくださいってことです。

例えばですが、はやしの場合は定期的に自然豊かな公園（福岡市内にある大濠公園）によく行くのですが、そこでいつも**「心地よいなぁ」** と感じるんですよね。

この**「心地よい」** という抽象的な表現を、自分なりに具体で解釈すると、このような構図になります。

心地よい  
↑なぜ？  
空気が透き通っていて木の自然な香りが漂う、晴れていて湿度が少なく快適、鳥の囀りが所々で聞こえて、自然の豊かさを感じる、水の流れる音がして、都会にいるのに田舎に来たような感覚になる

こんな感じで、自分が「心地よい」と思った理由を深掘りしていくんです。そうすると、私が「心地よい」と思う空間は、「自然豊かな環境」という言葉に言い換えられるんですよね。

人によっては、自然豊かな環境＝心地よい、とは限らない方も一定数いらっしゃいます。むしろ騒がしい環境が心地よいと思う人も。

シンプルライフを送りたいのであれば、まずは自分がどんな環境が好きなのか？を抽象的で良いので、場面毎に振り返ってみてください。はやしの場合は、

**人混みが嫌い / 自然豊かな場所の方が好き / コンパクトに生活できる環境が、時間を有意義に使える / 交通の便がよく、気軽に東京に行ける**

こんな感じで、自分にとっての好きな環境を書き出した結果、**「やっぱり福岡だよね」** という結論に至り、自分が理想とする生活、つまり、日常におけるあらゆる物事を効率化し、自分にとって本当に好きなことに注力できる時間を捻出することができたわけです。

これが「心地よい場所が良い」とか「都会は嫌だ」とか、抽象的な表現では「福岡に住む」という結論は導き出せないわけです。

**日頃から抽象的な表現を使いがちな人は、「なぜそう思ったのか？」の具体を出す訓練を意識してみましょう。**
