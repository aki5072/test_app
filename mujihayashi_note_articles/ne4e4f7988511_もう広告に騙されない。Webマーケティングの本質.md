# もう広告に騙されない。Webマーケティングの本質

先日、以下のようなポストをしました。

> YouTubeがサムネで決まるのも、ブログがタイトルで決まるのも、ショートムービーが冒頭2秒で決まるのも全部本質は一緒で。要は人間が直感的に「あ、これ続き見たい！」と思える誘導ができるか？ってことなのよね。この本質に気づけるか、気づけないかでマーケターの素質問われます。
> 
> — シンプリストはやし (@muji_hayashi) [November 29, 2024](https://twitter.com/muji_hayashi/status/1862488091304612301?ref_src=twsrc%5Etfw)

この140文字の中に**「Webマーケティングの本質」** を詰め込んでいます。

ちなみに今日の記事はタイトルにもある通り、あなたが今後**「広告に騙されない」ために知って欲しい本質** をお伝えしますので、SNS運用やマーケティングに携わっていなくても、お役立ていただける内容になっていますのでご安心ください。

日頃からWebサイトやSNSを見ていると、ふと広告をクリックしてしまう、ってことはありませんか？他にも、たまたまおすすめに流れてきたYouTuberの動画を見てしまったり、InstagramのリールやTikTokでつい指を止めて最後まで見てしまったり。

これにはマーケター側が「あえて」目に留まるように仕掛けた結果だと知っていれば、**「こんな広告クリックして、無駄な商品買わなくて済んだのに…」「ついYouTubeを見て2時間無駄にすることなかったのに…」** と後悔が減ると考えています。

ってことで、早速答えです。Webマーケティングの本質とは、

**ユーザーが無意識に行動するクリエイティブが作れるか？**

※クリエイティブとは？…広告の画像であったり、YouTubeのサムネイルであったり、ユーザーが最初に目に触れるもの、と捉えていただけると。

これに尽きます**。「無意識に行動してしまうってどういうこと？」** と疑問に思っている方もいると思うので、具体的な例をお伝えすると。

例えばSNSを見ていると広告が流れてきますよね。**その広告を「つい」クリックしてしまう…これが「無意識に行動する」という意味です。**

無意識に行動してしまったのは、「広告で紹介されている商品が良さそうだった」とか、転職系ならちょうど今転職を考えていて、「動くなら今だ！」みたいな背中を押してもらえる感じがしたとか。

この場合は、Webマーケ業界では無意識レベルで行動を誘発できるクリエイティブは「良いクリエイティブ」と評価されます。

一方で、無意識レベルで行動してしまう文章にはある程度法則があって…例を出すと、「これヤバイ」とか「見ないと損する」とか、不安を煽るような文言ですね。

この場合、確かに無意識レベルでは行動してしまうものの、冷静に考えてみると「全然大した商品ではないな」とか、「YouTubeの動画、思った以上に面白くないな」とか、**ユーザーの期待値を下げてしまう結果** となります。

これが俗に言う、流行りのクリエイティブを真似しただけの、本質を掴めていない**「悪いクリエイティブ」** です。

良いクリエイティブって、本当にユーザーの悩みにぶっ刺さる文言であったり、画像であったり、広告をクリックした後のLP（ランディングページ）の設計も見事なんですよね。

しっかりユーザーの悩みを理解しているからこそ、綺麗に購買までつなげていますが….先ほどお伝えした**「悪いクリエイティブ」は、広告をクリックした先の設計がまるでできていない。**

みなさんも広告をついクリックしようとした時は、こ**の話を思い出して「良いクリエイティブなのか、悪いクリエイティブなのか」を吟味すると、広告に騙されないネットリテラシーの高い人間になれます。**

ちなみにSNSやブログ、YouTubeを運用している方にお伝えすると、巷で言われている**「ショートムービーは2秒で決まる！」も本質は同じ** です。

要は人間が無意識に行動してしまう時間が**「2秒」** なので、その2秒の間に、無意識で行動してしまう要素を詰め込みましょうって話なわけで。

これがブログやYouTubeなるとサムネイルやタイトルになりますし、LP（ランディングページ）であればファーストビュー（最初に画面に表示されるエリア）になります。

こう考えてみると、マーケティングは心理学にも関連してくるので、本当に面白い分野ですよね。みなさんもマーケティングにぜひ興味を持っていただけると嬉しいです！
