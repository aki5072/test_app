# noteで「飯が食える」先にあったもの？

![eyecatch](https://assets.st-note.com/production/uploads/images/221908402/rectangle_large_type_2_3ac784517a9f2ff31cafc19f7e8bd42c.png?fit=bounds&quality=85&width=1280)

Xアカウント「noteで飯を食っていく」を開設したのが今年の5月末。早いもので、気づけばもう10月ですね。

アカウントを開設してから早4ヶ月が経ちましたが、この判断は間違ってなかったな、と思います。まずは実績面を振り返ると、Xのフォロワー数は0→1,400人になり、メンバーシップの会員数も、半年間20人だったのが、今では120人を超えるように。

ちなみに今月は値上げもしていますからね？500円→1,000円に値上げしても毎日会員数は増えているので、シンプルに先月の倍の売上が確定しています。

そんな感じで、名実ともに「noteで飯が食える」ようになったかな？と感じる今日この頃ですが、noteを伸ばしたいなら、よくこんな言葉を耳にしますよね。

**「noteを伸ばしたいなら、生活を見せよう」**

ってことで、今日は名実ともに「noteで飯が食える」ようになったはやしの生活の変化と言いますか、「noteで飯が食えるようになったら、こんな生活が送れるんだよ」というリアルを見せていこうかな、と。

結論から先にお伝えすると、

**無駄なことを楽しむ時間が増えました。**

はやしは元々「シンプリスト」として発信しており、シンプリストの定義としては、以下のように定めています。

**「身の回りのモノ、コトを効率化し、自分にとって本当に"好きなこと"に注力する」**

はやしがnote副業に目をつけたのも、まさに「効率化」という視点です。これまではInstagramやYouTubeなど、シンプルに「SNSを伸ばす」ことだけに注力していましたが、いざマネタイズ、となると…。

Instagramの場合は毎日ストーリーを投稿し、そこでアフィリエイトを訴求→アフィリエイトの案件が終了すれば、それこそ稼げなくなったり。YouTubeも、確かに動画を出せば出すほど広告収入は手に入るけど、ネタも擦りまくると視聴者が飽きてしまう。

**そこで出会ったのが、noteという存在です。**

h特にメンバーシップであれば、サブスク型なので毎月一定の会員数がいれば、月初時点で売上が確定している。仮に500円のメンバーシップで100人の会員がいれば、月初時点で50,000円の売上が確定している。

こんなに「効率的」に稼げる副業はないよね、という判断で、昨年の11月にメンバーシップを始めました。

そこから紆余曲折あり、100人まで伸ばした過程は以下noteで触れているので今回は言及しませんが、今ではnoteだけでも、直近3ヶ月で10,000円→80,000円の売上を立てることができています。

本題に戻りますが、noteだけでも10万円近い金額を得られるようになった今、生活においても変化がありました。

それこそ、昨年までは本業もやりつつ、副業では業務委託のマーケティング業務に勤しみつつ、マネタイズに繋がらないInstagramのリールを投稿し、たまにもう1つの副業である、完全成果報酬型の転職キャリアアドバイザーをやる日々。

こんなに動いているのに、特にSNSからは直接的な売上が保障されているわけでもなく、転職のキャリアアドバイザーに関しても、自分が決めないと報酬が入らないので、まさに**「タダ働き」** の比率が増えていったのです。

こんなに動いているのに、収入が入らない。そんな状況を変えたくて、

**「結局自分でコントロールできるマネタイズをやった方がいいよね」**

という考えもあり、noteを始めた、という背景もあります。

そして実際に今では？月初時点で5万円近い売上が確定している安心感もあり、

**「本業も合わせると十分稼いでいるから、副業そんなに動かなくてもいいよね」**

という、まさにシンプリストの考えである「効率的に暮らし、好きなことに注力できる人生」を手に入れました。

SNSも、平日はnoteを売りたいので本業をやりつつPCでXを更新していますが、土日はスマホを見る時間が減りました。ドライブを楽しみ、ちょっとしたプチ旅行や、散歩を楽しむようになりました。

スマホでは味わえない、カメラならではの質感。そして、写真しか撮れないからこそ、ネット断ちを行い、デジタルデトックスになるその瞬間。

**こういった、何気ない日常を愉しむ、という行為こそ、さらにnoteを伸ばすヒントが閃いたり、まさに「無駄」とも思える行為を楽しめる、そんな日々を手に入れることができたと思っています。**

一方で、いきなりこんな生活が手に入るとは思ってないですし、実際、はやしもXアカウントやnoteを3ヶ月コミットして伸ばしたので、一定の負荷は必要だと思っています。

実際、今年の6〜7月は土日も毎日のようにXの更新やnoteをずっと綴っていましたからね…とはいえ、はやしはこのフェーズは卒業かな、という感じです。

売上は増えていますが、実際にXやnoteに費やしている時間は減っていますからね。1日30分程度、Xやnoteに注力する時間だけ確保して、あとは勝手に稼いでくれています。

だからこそ、今お伝えしたような**「無駄な時間」** を愉しむことができるんです。これこそ、noteで飯を食っている人の**「リアル」** と言えるでしょう。
